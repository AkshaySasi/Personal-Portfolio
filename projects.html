Unveiling the Heart-Brain Connection: An Analysis of HRV in Cognitive Performance

MCGT is an innovative terminal-based Python application that converts finger gestures representing Morse code into English alphabets in real-time. Designed with accessibility in mind, this project helps bridge communication gaps through computer vision and machine learning.

Key Features

  • Real-time gesture recognition with 95% accuracy
  • Customizable sensitivity settings for different users
  • Multi-user support with profile saving
  • Export translation history for analysis
  • Interactive tutorial mode for learning Morse code
Python 3.8 OpenCV NumPy Pandas Scikit-learn

Challenges & Solutions

Developing robust gesture recognition required extensive testing with different lighting conditions and hand sizes. The biggest challenge was minimizing false positives while maintaining real-time performance. This was solved by implementing a dual-validation system that cross-checks gestures with temporal patterns.

  • Eclipse and planetary alignment simulations

This simulation has been adopted by three local schools for their astronomy programs. Teachers report a 40% improvement in student comprehension of orbital mechanics concepts compared to traditional teaching methods.
